성과등록(국내표준) 화면 도움말
표준화기구명
해당 표준의 제·개정한 표준화기관을 드롭박스 내에서 선택합니다.
1) 한국산업표준(KS): 드롭박스 내의 표준화기관 중 ‘국가기술표준원’ 또는 ‘국립전파연구원’ 선택 방송통신표준의 경우, ‘국립전파연구원’ 선택
2) 단체표준(SPS): 드롭박스 내의 표준화기관 중 ‘기타’ 선택 후 제정단체명 입력
3) 정보통신단체표준(TTAS): 드롭박스 내의 표준화기관 중 ‘한국정보통신기술협회’ 선택 *정보통신기술보고서(TTAR)은 표준이 아닌 표준화 관련 정보 제공 문서로, 연구성과에서 제외
표준번호
해당 표준화기구에서 부여한 표준번호를 입력합니다.(띄어쓰기 및 하이픈(-)기호 포함)
표준명
해당 표준 연구성과의 국문명을 정확하게 입력합니다.
제/개정 구분
승인된 표준의 제·개정 여부를 선택합니다.
제안기관
해당 표준을 제안한 과제의 참여기관의 이름을 입력합니다(복수입력 가능).
제안자
해당 표준을 제안한 과제의 참여연구자의 이름을 입력합니다(복수입력 가능).
표준채택(승인) 날짜
표준문서 표지에 명시된 표준 제·개정 승인날짜(YYYYMMDD)를 입력하는 항목으로, 우측의 달력 아이콘 클릭 후 해당 날짜를 선택합니다.
표준기술 요약
해당 표준기술의 내용을 요약하여 입력합니다(개조식, 500자 이내).
기여율
해당 표준의 제·개정을 위한 과제의 기여 비율을 입력합니다. 최대 합계 100을 기준으로 하며, 소수점 아래 한 자리까지 숫자로 입력 가능합니다(예: 100.0 또는 90.5).
증빙자료 링크
최종 발간된 표준을 확인할 수 있는 링크를 입력합니다(복수입력 가능).
첨부파일
① 최종 발간되어 표준번호가 기재된 표준 전자문서 원문, 및 ②KS와 SPS의 경우 표준 제·개정 신청서(날인본)을 첨부합니다.
­② 반드시 최종 발간된 표준 원문 첨부(초안 등 표준번호가 정확하게 기재되지 않은 문서는 증빙자료로 불인정됨)

표준화기구명
국제표준화를 추진하는 표준화기구명을 드롭박스 내에서 선택합니다.
표준화분과
해당 표준을 개발하는 표준화 기술분과위원회명을 WG(working group, 작업반)까지 번호로 입력합니다.
- 예) ISO/TC1/SC2/WG3
표준번호
해당 표준안에 부여받은 공식 표준번호를 전체 입력합니다.
- 예) 신규제안 채택: ISO NP 7683
승인된 국제표준안: ISO DIS 24281, IEC CDV 60001
발간표준: ISO 20468-7:2022, ISO/IEC 123456:2022
표준명
해당 표준 연구성과의 영문명을 full name으로 입력합니다.
- 예) Underwater Communication Technologies for IoT
제/개정 구분
해당 표준의 제정 또는 개정 여부를 선택합니다.
표준특허 추진 여부
표준특허 추진에 해당하는 경우, 박스 체크 후 표준특허명을 입력합니다.
표준성과 개발단계
해당 표준 연구성과의 개발단계를 드롭박스 내에서 각각 선택합니다.
- 예) ISO의 ‘신규제안 채택’에 대한 표준 연구성과를 입력하고자 할 경우, 첫번째 드롭박스에서 [ISO] 선택, 두번째 드롭박스에서 [AWI] 선택
표준제안 문서번호
해당 표준화기구 기술분과위원회에서 회람된 공식 표준제안문서상에 기재된 공식 문서번호를 입력합니다.
표준제안 날짜
공식 표준제안문서상에 기재된 표준제안 날짜를 입력하는 항목으로, 우측의 달력 아이콘 클릭 후 해당 날짜를 선택합니다.
표준제안 문서 제목명
공식 표준제안문서상에 기재된 영문 표준명(Title)을 입력합니다.
표준기술 요약
해당 표준 연구성과에 대한 기술내용을 요약하여 개조식으로 입력합니다(500자 이내).
증빙자료 링크1
증빙자료 첨부1
해당 표준화기구에서 회람된 공식 표준제안문서를 첨부합니다.
표준채택(승인) 문서 번호
1. 신규제안채택단계
해당 표준화기구 기술분과위원회에서 회람된 NP투표 결과문서로, 문서에 기재된 문서번호를 입력합니다.
- (ISO, ISO/IEC JTC1) N 943
- (IEC) 110/1368/RVN
2. 승인된 국제표준안 단계
해당 표준화기구 기술분과위원회에서 회람된 국제표준안 투표 결과문서로, 문서에 기재된 문서번호를 입력합니다.
3. 발간표준 단계
최종 발간을 위해 중앙사무국에 제출한 최종국제표준안의 공식 문서번호를 입력합니다.
표준채택(승인) 문서 제목명
1. 신규제안채택단계
NP투표결과 공식문서 상에 기재된 영문 표준명(Title)을 입력합니다.
2. 승인된 국제표준안 단계
해당 표준화기구에서 회람된 국제표준안 투표 결과문서에 기재된 영문 표준명(Title)을 입력합니다.
3. 발간표준 단계
최종 발간된 표준문서상에 기재된 영문 표준명(Title)을 전체 입력합니다.
표준채택(승인) 날짜
1. 신규제안채택단계
2. 승인된 국제표준안 단계
국제표준안 투표 결과문서에 기재된 표준채택(승인) 날짜를 입력하는 항목으로, 우측의 달력 아이콘 클릭 후 해당 날짜를 선택합니다.
- (ISO, ISO/IEC JTC1) Form13(Report of voting on ISO/DIS)의 Date
- (IEC) RVC(Result of voting on CDV)의 Circulation Date
3. 발간표준 단계
최종 표준 발간일자를 입력하는 항목으로, 우측의 달력 아이콘 클릭 후 해당 날짜를 선택합니다.
표준 발간일자는 각 표준화기구별 웹사이트에서 표준번호 검색 시 확인 가능합니다. * 최종 국제표준안은 22년에 제출되었으나 발간이 23년에 예정된 경우 23년 연구성과에 해당
표준화 회의명
1. 신규제안채택단계
(표준화회의를 통해 채택(승인)된 경우), 해당 표준 연구성과가 채택(승인)된 표준화 회의명(회차 포함)을 입력합니다.
2. 승인된 국제표준안 단계
(표준화회의를 통해, 채택(승인)된 경우), 해당 표준 연구성과가 채택(승인)된 표준화기구의 회의명(회차 포함)을 입력합니다.
3. 발간표준 단계
최종 발간이 승인된 표준화기구의 회의명(회차 포함)을 입력합니다.
증빙자료 링크2
1. 신규제안채택단계
해당 표준화기구에서 채택(승인)되었음을 확인할 수 있는 회람된 NP투표결과 공식문서 혹은 위원회 결의안문서의 URL 주소를 입력합니다.
2. 승인된 국제표준안 단계
해당 표준화기구에서 채택(승인)되었음을 확인할 수 있는 회람된 국제표준안 투표결과 공식문서 혹은 위원회 결의안문서의 URL 주소를 입력합니다.
3. 발간표준 단계
최종 발간된 표준문서를 확인할 수 있는 URL 주소를 입력합니다.
증빙자료 첨부2
1. 신규제안채택단계
해당 표준화기구에서 채택(승인)되었음을 확인할 수 있는, 회람된 NP투표결과 공식문서 혹은 위원회 결의안 문서를 첨부합니다.
2. 승인된 국제표준안 단계
해당 표준화기구에서 회람된 공식 표준채택(승인)문서를 첨부합니다.
3. 발간표준 단계
최종 발간(Publication)된 표준문서(IS)를 첨부합니다.
제안기관
해당 표준을 제안한 과제 참여기관명을 국문 full name으로 입력합니다. 참여기관이 복수인 경우 복수입력이 가능합니다.
제안자
해당 표준을 제안한 과제 참여연구자명을 입력합니다. 과제 참여 연구자가 복수인 경우 복수입력이 가능합니다.
프로젝트 리더/에디터 여부
표준 제안자가 표준을 제안하는 프로젝트 리더(ISO, JTC1) 또는 에디터(IEC, ITU)에 해당하는지 여부를 체크합니다.
제안자와 프로젝트 리더/에디터에 해당하지 않는 경우 ‘아니오’ 선택 후 프로젝트 리더/에디터의 이름을 입력합니다.
기여율
해당 표준의 제·개정을 위한 과제의 기여 비율을 입력합니다.
최대 합계 100을 기준으로 하며, 소수점 아래 한 자리까지 숫자로 입력 가능합니다.(예: 100.0 또는 90.5)
